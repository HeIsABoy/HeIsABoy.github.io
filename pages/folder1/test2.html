# CSS

# 样式表语言 看一下flex

# 1 基本样式

p {
  color: red;
}

打开 index.html 文件，然后将下面一行粘贴到文档头（也就是 <head> 和 </head> 标签之间）。

下边两句写到html中

<link href="styles/style.css" rel="stylesheet">
<--!字体设置-->
 <link href="https://fonts.font.im/css?family=Open+Sans" rel="stylesheet" type="text/css">

# 2 基本设置

font-size: 字体大小
font-family：字体类型
background-color： 背景颜色
width：这一项的宽度
margin：外边框宽度
padding：内边框宽度
border： 边框样式
text-align: 设置 h1、h2、h3 元素的文本对齐方式：
color: 为不同元素设置文本颜色
text-shadow: 基础的文本阴影效果
line-height: 设置以百分比计的行高
letter-spacing: 设置 h1 和 h2 元素的字母间距

# 3 flex布局

控制父盒子：

flex-direction: column; //定义主轴方向
justify-content: space-around; //主轴方向上元素的排列方式
align-items:center ;//侧轴方向上元素的排列方式
flex-wrap: wrap; //换行方式
align-content //多行存在时的换行方式 通过侧轴控制
flex-flow:column wrap;//相当于flex-direction 与 flex-wrap结合 哪个在后边哪个起作用
algin-content：//多行排列方式

单行找algin-item 多行找algin-content

控制子盒子：

flex:子项目占的份数
algin-self:控制子项自己在侧轴的排列方式（比如一排有三个盒子，控制其中一个盒子的排列方式）
order属性定义子项的排列顺序（前后顺序 比如5个盒子顺序是12345 我要改成21345）
flex：比如说一个大盒子中有三个盒子元素，然后其中两个在两端固定了，中间的那个可以随着大盒子的大小变化而变化
